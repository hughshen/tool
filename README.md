# Toolset

> A toolset build with Element UI.

## Environment

`Node >= 6`

## Start

 - Clone or download this repository
 - Enter your local directory, and install dependencies:

``` bash
yarn install
```

## Develop

``` bash
# serve with hot reload at localhost:8010
yarn run dev
```

## Build

``` bash
# build for production with minification
yarn run build
```
